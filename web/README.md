# Axobase Web 观测台

可视化界面用于观察 Axobase AI Agent 的进化过程。

## 快速开始

### 1. 安装依赖
```bash
cd web
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 打开浏览器
访问 http://localhost:3000

## 功能

- 📊 实时种群统计（存活数、余额、繁殖/死亡事件）
- 👥 Agent 列表（状态、余额、策略、生命周期阶段）
- 🧬 基因表达分布可视化
- 📜 事件日志
- 🎮 控制面板（开始/暂停/重置）

## 技术栈

- Next.js 14
- React + TypeScript
- Tailwind CSS
- Recharts
